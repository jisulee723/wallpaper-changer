# wallpaper-changer

바탕화면 변경 프로그램

## How to use?

1. exe 파일을 받아 프로그램을 실행

2. 시작 시 값을 입력해 북부기술교육원 식단표로 배경화면 설정 혹은 TG삼보 기본 바탕화면으로 변경
