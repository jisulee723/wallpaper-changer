# wallpaper-changer

바탕화면 변경 프로그램

## How to use?

1. exe 파일을 받아 프로그램을 실행
2. 북부기술교육원 식단표와 TG 삼보 기본 바탕화면 중 선택
3. 바탕화면 적용 방식을 선택
